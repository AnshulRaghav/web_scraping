# web_scraping
data of odi players list and the runs made by them each year have been scrapped using python library beautifulsoup
index.ipnyb - jupyter notebook file where the main code lies
final.csv - csv file containg list of players and their country's alongwith the runs that they have made each year
runs_by_year.csv - csv file containg list of total runs each year grouped by player names
runs_by_year_id.csv - csv file containg list of total runs each year grouped by player id's
player_list.csv - csv file containg list of total runs made by each player along with their id's and country's
